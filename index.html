Comprendre les bases de HTML et de CSS

L'objectif de ce chapitre était de nous faire comprendre HTML/CSS.
Pourquoi apprendre le HTML?

-Il s'agit de deux langages indispensables pour tout projet.
-Ils sont deux standards qui ne possédent pas de concurrents.
-Apprendre le HTML et le CSS, un bon début pour apprendreà développer... Mais également pour comprendre son propre site et les professionnels du secteur!

Pourquoi indenter son code?

Indenter permet de créer des retraits en début de ligne de façon conhérente et logique. En indenter, on a un code plus clair et ligible. il s'agit d'un retrait vers la droite réalisé avec la touche Tab du clavier. Identer son code signifie que l'on va ajouter un retrait à droite de même taille en fonction du niveau de parenté des Elements.

Pourquoi commenter son code?

Il s'agit des lignes de texte que l'on va écrire au milieu de notre code afin de donner des indications sur ce que fait le code. Ils sont invisibles pour les visiteurs et ne servent qu'au développeur créant le code. Ils sont utils dans le cas d'un gros projet afin de se rappeler pourquoi nous avons écrit tel ou tel code ou encore pour se repérer dans le code. Ils permettent si nous travaillons en équipe de permettre aux autres développeurs de mieux comprendre le code. Ils peuvent être monoligne ou multilignes
